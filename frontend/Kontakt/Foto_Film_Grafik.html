Foto, film eller grafik?

Ljungby Simsällskap söker dig som ideellt vill hjälpa oss att berätta klubbens historia och lyfta hur klubben syns utåt.

Foto, film och grafik Flera personer kan hjälpa till Du behöver inte vara medlem

Det här vill vi bli bättre på

Vi vill bli bättre på att visa träningar, tävlingar, gemenskap, utveckling och alla de viktiga ögonblick som finns i föreningen.

Därför söker vi gärna någon eller några som vill hjälpa till med foto, kortare filmklipp, enklare innehåll till sociala medier eller ibland även enklare grafiskt material.

Vi söker känsla, inte perfektion

Det handlar inte bara om att ta en bild. Vi söker gärna dig som har känsla för bild, ögonblick, vinklar, uttryck och helhet.

Du kanske också tycker om enkel redigering som kan göra materialet tydligare, varmare och mer levande.

Det kan handla om

Vid träning eller tävling

Fånga några bra bilder, korta klipp eller stämningsfulla ögonblick när det passar.

Till sociala medier

Hjälpa till med enklare innehåll som visar glädje, utveckling och gemenskap i klubben.

Enklare grafik

Ibland kan det även handla om enklare grafiskt material som gör information tydligare och mer levande.

Det här kan vi göra tillsammans

Det här behöver inte vara en enda person. Tvärtom kan det vara värdefullt om vi blir några stycken som kan hjälpa till vid olika tillfällen, turas om eller ibland fånga samma event från olika håll.

Flexibelt upplägg: du kan hjälpa till ibland, tillsammans med andra eller bara vid tillfällen som passar dig.

Bra att veta

Du behöver inte vara medlem i klubben för att höra av dig. Det viktigaste är att du vill bidra med din kreativitet och hjälpa oss stärka LSS.

Det mesta vi söker hjälp med sker på hemmaplan. Hjälp vid bortatävlingar eller andra event är ett extra plus, men inget krav.
